# دليل تشغيل مشروع SHOW.TV على VS Code

أهلاً بك! اتبع هذه الخطوات البسيطة لتشغيل المشروع على جهازك (Ubuntu) باستخدام VS Code.

## المتطلبات المسبقة (Prerequisites)
تأكد من تثبيت البرامج التالية على جهازك:
1. **PHP** (الإصدار 8.0 أو أحدث)
2. **Composer** (مدير حزم PHP)
3. **Node.js** & **NPM**
4. **MySQL** (قاعدة البيانات)

## خطوات التشغيل (Step-by-Step)

### 1. فتح المشروع
* قم بفك الضغط عن ملف `showtv_project.zip`.
* افتح برنامج **VS Code**.
* اختر **File** > **Open Folder** واختر مجلد `showtv`.

### 2. تثبيت المكتبات (Dependencies)
افتح التيرمينال داخل VS Code (اختصار: `Ctrl + ~`) ونفذ الأوامر التالية بالترتيب:

```bash
# تثبيت مكتبات PHP
composer install

# تثبيت مكتبات الواجهة الأمامية
npm install
```

### 3. إعداد البيئة (Environment Setup)
قم بإنشاء ملف الإعدادات `.env` وتوليد مفتاح التشفير:

```bash
# نسخ ملف الإعدادات الافتراضي
cp .env.example .env

# توليد مفتاح التطبيق
php artisan key:generate
```

### 4. إعداد قاعدة البيانات (Database)
1. تأكد من تشغيل MySQL وإنشاء قاعدة بيانات جديدة فارغة (مثلاً باسم `showtv_db`).
2. افتح ملف `.env` في VS Code وقم بتعديل بيانات الاتصال بقاعدة البيانات:

```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=showtv_db
DB_USERNAME=root       <-- اسم المستخدم الخاص بك
DB_PASSWORD=           <-- كلمة المرور الخاصة بك
```

3. بعد الحفظ، نفذ الأمر التالي لإنشاء الجداول وإضافة البيانات التجريبية:

```bash
php artisan migrate --seed
```

### 5. تشغيل المشروع (Run Application)
أنت الآن جاهز! نفذ الأوامر التالية لتشغيل السيرفر:

```bash
# تشغيل سيرفر Laravel (في نافذة تيرمينال)
php artisan serve
```

ستظهر لك رسالة تفيد بأن السيرفر يعمل على رابط مثل: `http://127.0.0.1:8000`
اضغط على الرابط لفتح الموقع في المتصفح.
